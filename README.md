# Typescript Three.js Webpack Starter

- Typescript support.
- Webpack
- Html plugin

## Usage

### Dev

```
$ yarn install
$ yarn dev
```

### Build

```
$ yarn build
```

### Serve Build File

```
$ yarn start
```

### Import images

read from `src/assets/images`

```
import image from '~/assets/images/something.png'
```

### Static images

read from `static/images`

```
<img src="/images/something.png">
```
